Klik Untuk Melakukan Pemesanan/Pembelian Produk/Jasa Berikut

Produk/Jasa
"Menjadi Front End Developer Kreatif" adalah penyelengaraan kegiatan pembelajaran yang dapat diikuti oleh peserta dengan berbagai profesi Laki-laki atau Perempuan dengan rentang usia 18 tahun - 50 tahun. Peserta akan mendapat materi bagaimana membuat website sehingga setelah mengikuti kelas ini peserta akan mampu membuat dan mengembangkan web dengan HTML + CSS + JAVASCRIPT + JQUERY + XML dan lainnya.

Pelaksanaan
Peserta dalam kota dapat mengikuti pembelajaran secara offline (datang langsung ke kelas) dan online
peserta luar kota mengikuti pembelajaran secara online
